## Short-forms

* DM -> Data Members
* IO -> Input-Output
* I/P or IP -> Input
* O/P or  OP -> Output
* ref-Id or ref-id -> Reference ID
* psvm -> public static void main(String args[])
* ref. -> reference
* obj -> object
* args -> arguments
